# Sankalp Digital Pathshala

Premium multi-page educational ecosystem website with a Node.js backend, MongoDB, Cloudinary uploads, AI assistant support, result checking, lead capture, and an admin dashboard.

## Setup

1. Run `npm install`.
2. Update `.env` with your private deployment values.
3. Run `npm start`.
4. Open `http://localhost:3000`.

The project keeps `.env.example` as the environment-variable reference. Never commit real credentials.

## Brand

The established Sankalp Digital Pathshala logo, favicon, and Sky Blue identity are preserved across the website.
